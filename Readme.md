# 🛍️ Nostra – E-Commerce Website

A modern **responsive e-commerce website** built using **HTML, CSS, and JavaScript**.
This project is inspired by the Nostra design and focuses on creating a **clean UI, smooth animations, and interactive shopping experience**.

✔ Modern responsive design
✔ Smooth animations and hover effects
✔ Product showcase layout
✔ Search functionality
✔ Interactive UI elements

✔ Clean and structured code
